# Matrixfamily

Nuestra jerarquía de clases para implementar las matrices será:
- Una matriz madre
- Dos matrices hijas (Densa y Dispersa)

## Installation

Add this line to your application's Gemfile:

    gem 'matrixfamily'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install matrixfamily
